# Importance of Exercise

每天都会产生新的神经元，比如海马体。锻炼比任何药物都起作用，来促进这些新神经元的产生和运作。

# Imposter Syndrome

感觉自己能力不足，将现有成果归因于只是运气好，认定下一次考试一定会失败并且被揭露是骗子。

如果被这种不够好的感觉所苦恼，就当成其他人也偷偷共享着这种感觉。

# The Value of Teamwork

右脑倾向质疑现状，帮助我们跳脱问题，保证思路的正确且符合常理。

左脑倾向维持现状，善于提供分析与积极的方法。

# How to Take a Test

考试开始后，先把所有题目看一遍，将目光停留在难题上，当思考遇到瓶颈时迅速将注意力转移到简单题目上，这时大脑将刚刚在专注模式下思考的难题，转移到发散模式去处理。到时候回过头来也许难题就有了新的方向可以尝试。

让大脑在不同模式下交互运作，也帮助我们避免定势效应。这也是为什么我们总是在走出考场后才能顿悟解题方法。

答题顺序与检查顺序的不同，可以让大脑有不同的视角重新审视作答，更容易找到错误。
